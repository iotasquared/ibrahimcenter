# The house system (evidence: profile mark, "A Story of Hearts" post, campus sign)

**Grounds:** ivory/cream, generously empty. **Ink:** warm near-black charcoal. **Thread:** antique gold — the interlace mark, ornamental knot rules, letterspaced small-cap details. **Type voice:** elegant serif caps (Trajan/Cormorant-like) for display; italic script for the humane word ("Shukr", "Eid Prayer", "Suhoor with"); letterspaced small caps for logistics. **Motifs:** tone-on-tone eight-point-star tessellations; laurel value-badges; illuminated-manuscript and astrolabe ornaments for sacred seasons; circle-framed portraits for people.

# The program registers (each verified against multiple flyers)

| Program / family | Ground & accent | Character |
|---|---|---|
| Institutional (About, film, sign) | Ivory + charcoal + gold | Editorial, maximal whitespace, laurels |
| Ramadan / sacred seasons | Deep forest green + gold | Arabic calligraphy (رمضان) as art, knot rules, ornament |
| Eid | Charcoal or green + gold | Astrolabe/concentric ornament, script + schedule |
| Umrah / journeys | Deep maroon + gold | Regal; Kaaba imagery; pattern bands |
| Menaissance | Espresso brown + cream | Stark letterspaced caps; own logomark "Reviving the Masculine" |
| Sisters' Caravan | Ivory + blush watercolor + olive-gold | Soft, script-forward, feminine |
| Little Stewards camps | Cream + playful pastel per camp + gold script | Hand-drawn, bubbly display type (Seerah Through Songs = teal; Fatima al-Zahra's Table = dusty rose) |
| Family Majlis | Emerald teal + manuscript art | Devotional-warm |
