# Ibrahim Islamic Center — Institutional Brain

The canonical memory of Ibrahim Islamic Center, Houston (618 Baca Street, Third Ward roots, founded by converts 20+ years ago). Everything true about the institution lives here as plain text; the public website is a projection built from it. The only editing interface is the AI consultant — see [CLAUDE.md](CLAUDE.md) for the constitution (one-interface law, approval law, privacy boundary, entity conventions).

```
brain/
  identity/      mission, messaging, voice, values      (changes ~never)
  place/         campus, parking, accessibility         (years)
  people/        one file per person, role-dated        (yearly)
  governance/    board, form routing, decisions         (yearly)
  programs/      one file per program, lifecycle-tracked (quarterly)
  giving/        donations, zakat policy                (quarterly)
  audiences/     who the institution serves + their needs
  history/       the story, timeline
  events/        append-only; the living record          (weekly)
  media/         asset manifest + consent tracking
  OPEN-QUESTIONS.md   every unverified fact, with owner — the content burndown
  DECISIONS.md        why things are the way they are
  REDIRECTS.md        slug renames (URLs are permanent)
site/            the static build — disposable by design (phase 0: pending)
workbook/        source content workbook (provenance)
```

**Status:** phase 0 — brain scaffolded from the July 2026 content workbook; site build next; launch target 2026-08-01.
